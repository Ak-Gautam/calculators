### Details

- Simple calculator
- Written in C language
- Runs in terminal / CMD Prompt
- Performs +, -, *, /
- Can calculate only two numbers
- Input format ``` a (op) b ``` e.g. 2 + 4
- Output format ``` a (op) b = res ``` e.g. 2 + 4 = 6
